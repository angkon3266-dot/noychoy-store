# On-page checklist (per product)

Run this list when publishing or revising any product:

1. **Title** contains the primary keyword naturally (material + type).
2. **Slug** is short, keyword-bearing, hyphenated — set once, never churn.
3. **Meta title** ≤ 60 chars, primary keyword first.
4. **Meta description** ≤ 160 chars with COD trust signal.
5. **Short description** opens with the primary keyword in the first sentence.
6. **Images**: primary image shows the piece worn or at flattering scale;
   every image has descriptive alt text (product file → SEO section).
7. **Category assignment** correct (feeds collection pages + filters).
8. **Tags** include occasion + material (drive internal filter URLs).
9. **Cross-links**: product body links its material file topics on-page where
   the theme supports it (e.g. care instructions block).
10. **Price/stock sanity**: availability correct — OG tags and the Meta
    catalog feed publish it.
